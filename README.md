# Customer Management System Documentation

## Features
- **User Authentication:** Users can register and log in securely.
- **Customer Database:** Store and manage customer information efficiently.
- **CRUD Operations:** Create, Read, Update, and Delete customers.
- **Search Functionality:** Easily search for customers based on various criteria.
- **Reports:** Generate reports for customer analysis.

## Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/himanshuparashar27/customer-management-system.git
   cd customer-management-system
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Set up environment variables:** Create a `.env` file and populate it with the required configurations.
4. **Run the application:**
   ```bash
   npm start
   ```

## Usage Guide
- To access the application, open a web browser and go to `http://localhost:3000`.
- Follow the prompts to register or log in.
- Use the dashboard to manage customers and access other features.

- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Authentication:** JSON Web Tokens (JWT)
